- TensorFlow
- Numpy
- Matplotlib
- Pandas
- Sklearn
- Unidecode

## Jupyterlab setup

- Install Anaconda
- Create new environnment
- Open environment terminal
- \>pip install tensorflow
- \>pip install pandas
- \>pip install sklearn
- \>pip install jupyter notebook
- ...
- Open jupyter notebook in that environment
